# 🤖 Task Manager Bot

Telegram-бот для управления задачами.  
Позволяет пользователю добавлять, просматривать и удалять задачи прямо в Telegram.

---

## 📌 Функциональность

- ➕ Добавление задач (`/add`)
- 📋 Просмотр списка задач (`/list`)
- ❌ Удаление задачи (`/delete`)
- 🧹 Очистка списка (`/clear`)
- ℹ️ Помощь (`/help`)

---

## ⚙️ Технологии

- Python 3
- Telegram Bot API
- библиотека `python-telegram-bot`
- JSON (для хранения данных)

---

## 📂 Структура проекта

- `main.py` — основной файл бота  
- `requirements.txt` — зависимости  
- `tasks.json` — хранение задач
